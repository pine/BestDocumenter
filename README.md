# BestDocumenter
Count your document contributions

## Requirements
### Mac OS X
First, you must install [Homebrew](http://brew.sh/).

```
$ brew install cmake curl gflags
```

### Ubuntu (14.04)

```
$ sudo apt-get install g++ cmake libgflags-dev libcurl4-openssl-dev
```

## Build

```
$ cmake .
$ make
```
